Workout API Documentation

Activity Endpoints

Endpoint Method Description Input Parameters Status Codes
/activity/ GET Hämtar alla aktiviteter. Returnerar en lista med aktiviteter. N/A
  • 200 OK - activities [{object}] - id {int} - name {string} - has_time {bool} - has_distance {bool} - distance_unit {string} - user {int}
  • 400 Bad Request: Ogiltiga query-parametrar.
/activity/ POST Sparar en ny aktivitet.
  • name (str, required)
  • has_time (bool, required)
  • has_distance (bool, required)
  • distance_unit (str, optional, 'km' eller 'm')
  • user (int, required)
  • 201 Created - activity {object} - id {int}
  • 400 Bad Request: Ogiltiga inparametrar.
/activity/{id} GET Hämtar en specifik aktivitet baserat på id.
  • id (int, required)
  • 200 OK - activity {object} - id {int} - name {string} - has_time {bool} - has_distance {bool} - distance_unit {string} - user {int}
/activity/{id} PUT Uppdaterar en specifik aktivitet baserat på id.
  • id (int, required)
  • name (str, optional)
  • has_time (bool, optional)
  • has_distance (bool, optional)
  • distance_unit (str, optional, 'km' eller 'm')
  • user (int, optional)
  • 200 OK - activity {object} - id {int} - name {string} - has_time {bool} - has_distance {bool} - distance_unit {string} - user {int}
  • 400 Bad Request: Ogiltiga inparametrar.
/activity/{id} DELETE Raderar en specifik aktivitet baserat på id.
  • id (int, required)
  • 204 No Content

Workout Endpoints

Endpoint Method Description Input Parameters Status Codes
/workout/ GET Hämtar alla workouts. Returnerar en lista med workouts. N/A
  • 200 OK - workouts [{object}] - id {int} - date {date} - activity {int} - user {int} - description {string} - time {duration} - distance {decimal} - rpe {int}
  • 400 Bad Request: Ogiltiga query-parametrar.
/workout/ POST Sparar en ny workout.
  • date (date, required)
  • activity (int, required)
  • user (int, required)
  • description (str, optional)
  • time (duration, optional)
  • distance (decimal, optional)
  • rpe (int, optional)
  • 201 Created - workout {object} - id {int}
  • 400 Bad Request: Ogiltiga inparametrar.
/workout/{id} GET Hämtar en specifik workout baserat på id.
  • id (int, required)
  • 200 OK - workout {object} - id {int} - date {date} - activity {int} - user {int} - description {string} - time {duration} - distance {decimal} - rpe {int}
/workout/{id} PUT Uppdaterar en specifik workout baserat på id.
  • id (int, required)
  • date (date, optional)
  • activity (int, optional)
  • user (int, optional)
  • description (str, optional)
  • time (duration, optional)
  • distance (decimal, optional)
  • rpe (int, optional)
  • 200 OK - workout {object} - id {int} - date {date} - activity {int} - user {int} - description {string} - time {duration} - distance {decimal} - rpe {int}
  • 400 Bad Request: Ogiltiga inparametrar.
/workout/{id} DELETE Raderar en specifik workout baserat på id.
  • id (int, required)
  • 204 No Content
